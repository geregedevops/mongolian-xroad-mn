## How to write FAQ entries

(Энэ хэсэг template-аас хасах — instructional comment.)

✅ **Сайн FAQ entry**:

- Question нь end-user-ийн өөрөө асуух хэлбэр (вооб не "What is X?" харин
  "Хэрэв X тохиолдвол яах вэ?").
- Хариулт нь 3-7 өгүүлбэр (хэт богино биш, хэт урт ч биш).
- Step-by-step list хэлбэр concrete actions-той.
- Хэрэв related doc байгаа бол link.

❌ **Муу FAQ entry**:

- Marketing tone (нөмжмөргийн speech биш).
- Очень общий ("read the documentation").
- Хариу-гүй (зөвхөн disclaimer / disclaimer-аар дүүрэн).

✅ **Сайн grouping**:

- Audience flow-аар (registration → login → daily use → security → tech).
- 5-10 question per group максимум.
